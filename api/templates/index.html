Welcome to FFBProject API.

An API for retrieving information about NFL players and performances.

API Endpoints

The base URL for all endpoints is http://127.0.0.1:5000/api

  • /players : Returns the list of all players in the database. Sample Response
  • /stats/<name>/<year> : Returns the season total stats of a specific player from a specific year. Sample Response
  • /stats/<name>/<year>/<week> : Returns the game stats of a specific player from a specific week. Sample Response
  • /top/<year>/<week> : Returns the stats of all players from a specific year and week ordered by fantasy points finish. Sample Response
  • /top/<year>/<week>/<pos> : Returns the stats of all players of a specific position from a specific year and week ordered by fantasy points finish. Sample Response

Notes

  • Only data between the years 2012 to 2021 and the weeks 1 to 17 (18 for 2021) can be retrieved.
  • In addition, data is only available for the positions of QB, RB, WR, and TE
