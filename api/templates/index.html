Welcome to FFBProject API

An API for retrieving information about NFL players and performances

API Endpoints

The base URL for all endpoints is http://127.0.0.1:5000/api

/players/<pos> : Returns the list of all players in the database.

  • Parameters: Pos
    • Pos (Optional) : The position of players to be returned. Options are QB, RB, WR, TE. If no position is specified, players of all four positions will be returned.
  • Sample Response

/stats/<name>/<year>/<week> : Returns the game stats of a specific player.

  • Parameters : Name, Year, Week
    • Name (Required) : The name of the player whose stats will be returned. Format as Firstname_Lastname.
    • Year (Optional) : The year from which to return stats. If year is not specified, the career total stats for the specified player will be returned.
    • Week (Optional) : The week of the season from which to return stats. If week is not specified, the season total stats of the specified year will be returned.
  • Sample Response

/top/<year>/<week>/<pos> : Returns the stats of players ordered by fantasy points finish.

  • Parameters : Year, Week, Pos
    • Year (Required) : The year from which to return stats
    • Week (Optional) : The week from which to return stats. If the week is not specified, the season totals for the specified player will be returned.
    • Pos (Optional) : The position to filter players by. The options are QB, RB, WR, and TE. If the position is not specified, the stats from all players will be returned.
  • Sample Response

Notes

  • Only data between the years 2012 to 2021 and the weeks 1 to 17 (18 for 2021) can be retrieved.
  • In addition, data is only available for the offensive skill-position players (QB, RB, WR, TE).
