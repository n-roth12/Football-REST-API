Welcome to the FFBProject API.

An API for retrieving information about NFL players.

API Endpoints

The base URL for all endpoints is http://127.0.0.1:5000/api

  • /players : Returns the list of all players in the database. Try endpoint
  • /stats/<name>/<year>/<week> : Returns the game stats of a specific player from a specific week. Try endpoint
  • /stats/<name>/<year> : Returns the season total stats of a specific player from a specific year. Try endpoint
  • /top/<year>/<week> : Returns the top fantasy player for a specific year and week including their stats. Try endpoint

Only data between the years 2012 to 2021 and the weeks 1 to 17 (18 for 2021) can be retrieved.

In addition, data is only available for the positions of QB, RB, WR, and TE
